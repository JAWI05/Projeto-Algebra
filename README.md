# 🚀 Browser Productivity Extension

> Projeto de Extensão desenvolvido durante o 4º semestre de Engenharia de Computação na ESEG.

Este projeto consiste em uma ferramenta projetada para otimizar o fluxo de trabalho e o gerenciamento de tempo diretamente no navegador. Criada para solucionar desafios comuns de foco, a extensão integra múltiplas funcionalidades em uma interface única e leve.

## 📌 Objetivo
O objetivo principal é mitigar distrações e centralizar ferramentas de produtividade (Timer, Tasks e Bloqueio) onde o trabalho acontece: no navegador.

## 🛠️ Habilidades & Tecnologias
- **Frontend:** HTML5, CSS3, JavaScript (ES6+).
- **APIs de Navegador:** `chrome.storage`, `chrome.alarms`, `chrome.tabs`.
- **Arquitetura:** Service Workers (Manifest V3), Content Scripts e Popup UI.
- **Engenharia de Software:** Persistência de dados local, Programação Assíncrona e UX Design.

## ✨ Funcionalidades
- **⏲️ Motor Pomodoro Integrado:** Timer customizável com persistência via Background Scripts.
- **📝 Gerenciador de Tasks:** Organização de prioridades sem troca de abas.
- **🚫 Filtro de Conteúdo:** Bloqueio de URLs (blacklist) para períodos de foco.
- **📊 Monitoramento de Tempo:** Estatísticas de navegação por domínio.
- **💾 Persistência:** Configurações mantidas entre sessões via Local Storage.
